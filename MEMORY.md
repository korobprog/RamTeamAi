# Memory

## Правила проекта

- Все новые пользовательские строки в интерфейсе и системных сообщениях записывать только в UTF-8. Не использовать запись через PowerShell/Python с проблемной кодовой страницей, если текст содержит кириллицу; лучше применять `apply_patch` или Unicode-escape.
- После правок с русским текстом обязательно проверять `src/` поиском по `?????`, `вЂ`, `Рќ`, `СЃ` и другим признакам mojibake.
- Если в локальном хранилище уже есть повреждённые сообщения с `????`, при загрузке сессий они должны мигрироваться в нормальный текст, а не показываться пользователю.
