# Memory

## Правила проекта

- Все новые пользовательские строки в интерфейсе и системных сообщениях записывать только в UTF-8.
- Если строка содержит кириллицу, не менять её через Python/PowerShell `write_text`, `.replace()` или here-string: в этой среде они уже несколько раз превращали русский текст в `????` или mojibake. Для таких правок использовать `apply_patch` или Unicode escape-последовательности.
- После каждой правки с русским текстом обязательно проверять исходники командой поиска по `?????`, `вЂ`, `Рќ`, `РЎ`, `СЃ`, `Р°`.
- Если в локальном хранилище уже есть повреждённые сообщения с `????`, при загрузке сессий они должны мигрироваться в нормальный текст, а не показываться пользователю.
- Перед финальным ответом по UI-правкам всегда запускать `npm run build`.
